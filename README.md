# Signal & Salvage

A text-forward incremental/clicker game built with **React + TypeScript + Vite**.

## Run locally

```bash
npm install
npm run dev
```

## Gameplay loop

- Manual scanning for early Signal
- Passive generator economy (6 tiers)
- Upgrades across Signal, Discovery Points, and Relays
- Findings (milestones) that reward DP
- Prestige reset into permanent Relays
- Local save/export/import with hard reset
